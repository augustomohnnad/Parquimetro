# Parquímetro
Este projeto tem como base a aplicação dos quatro pilares da Programação Orientada a Objetos (POO):
Abstração, Encapsulamento, Herança e Polimorfismo.

## Objetivo
O objetivo deste exercício foi praticar os conceitos de JavaScript e POO, além de HTML e CSS, utilizando apenas o front-end para interação com o usuário.

## Funcionamento
1. O projetofunciona da seguinte forma:
2. O usuário digita no teclado virtual o valor pretendido.
3. O JavaScript calcula o tempo correspondente, garantindo que não ultrapasse 120 minutos (2 horas).
4. O usuário confirma o pagamento, finalizando a operação e concluindo a usabilidade.

## Possíveis melhorias
Criação de tickets para impressão.
Melhorias na estrutura, adicionando banco de dados e integração com APIs.
Introdução de novas tecnologias, como React, Next.js, Bootstrap ou Tailwind CSS.

## Tecnologias estudadas
JavaScript (POO, DOM, eventos)
HTML5
CSS3

## Nota
Este exercício tem como finalidade a prática autodidata, sendo parte do estudo para a formação em Desenvolvimento Full-Stack na EBAC(Escola Britânica de artes criativas & tecnologias).
